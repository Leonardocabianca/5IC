# Chat
ho realizzato una chat udp.

## UDP
Per realizzare questo progetto è stato utilizzato il protocollo UDP.
Facendo uso del Multicast dove è presente un indirizzo ip al quale ogni client è in ascolto e può iviare messaggi.

# Grafica
Ho creato la grafica facendo uso di java FX con lo scenebuilder , quindi modificando il file XML creato inizialmente da Netbeans

# Server
Il server serve semplicemente per registrare gli utenti che accedono alla chat su un Database.
il client all'avvio invia tramite protollo UDP il proprio nome al server e si connette 

# Librerie
